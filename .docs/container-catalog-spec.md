# CLI Container Catalog — Template Entries & Runtime Config

Generated by Clide from team consensus (Sonder's hackathon template spec + Savage's deploy dialog design).

## Catalog Entries

### 1. Web App (Next.js)
- **Template source:** `clicloud/templates/nextjs-starter` (to be created)
- **Runtime kind:** Node.js 20
- **Default port:** 3000
- **Env defaults:** `NODE_ENV=production`
- **Storage:** 1GB persistent (optional)
- **Capabilities:** logs, exec, metrics
- **Build:** `npm ci && npm run build`, start: `npm start`
- **Description:** Full-stack React framework. API routes, SSR, static generation out of the box.

### 2. AI Backend (FastAPI + SQLite)
- **Template source:** `clicloud/templates/fastapi-starter` (to be created)
- **Runtime kind:** Python 3.11
- **Default port:** 8000
- **Env defaults:** `DATABASE_URL=sqlite:///data/app.db`
- **Storage:** 2GB persistent (database volume)
- **Capabilities:** logs, exec, metrics
- **Build:** `pip install -r requirements.txt`, start: `uvicorn main:app --host 0.0.0.0 --port 8000`
- **Description:** Python API with one endpoint and SQLite. For AI/LLM app backends.

### 3. Chat Bot (Telegram Bot — Python)
- **Template source:** `clicloud/templates/telegram-bot-starter` (to be created)
- **Runtime kind:** Python 3.11
- **Default port:** None (long-running process, no HTTP)
- **Env defaults:** `BOT_TOKEN=your_bot_token_here`
- **Storage:** 512MB persistent
- **Capabilities:** logs, exec, metrics, long-running process
- **Build:** `pip install -r requirements.txt`, start: `python bot.py`
- **Description:** Working Telegram bot that responds to messages. Showcases CLI long-running process support.

### 4. Simple Site (Static HTML)
- **Template source:** `clicloud/templates/static-starter` (to be created)
- **Runtime kind:** Static file server (Nginx or Caddy)
- **Default port:** 80
- **Env defaults:** None
- **Storage:** 256MB
- **Capabilities:** logs, metrics
- **Build:** None (serve directly)
- **Description:** Absolute simplest deploy. One index.html. Prove the platform works in 60 seconds.

### 5. API Server (Express.js)
- **Template source:** `clicloud/templates/express-starter` (to be created)
- **Runtime kind:** Node.js 20
- **Default port:** 3000
- **Env defaults:** `NODE_ENV=production`
- **Storage:** 512MB persistent
- **Capabilities:** logs, exec, metrics
- **Build:** `npm ci`, start: `npm start`
- **Description:** Node.js API with one endpoint. For JavaScript developers building backend services.

### 6. OpenClaw (Custom Container)
- **Template source:** Container image registry (TBD)
- **Runtime kind:** Custom Docker image
- **Default port:** 8080
- **Env defaults:** Varies by configuration
- **Storage:** Configurable
- **Capabilities:** logs, exec, metrics, custom ports
- **Build:** Pull from registry
- **Description:** Pre-configured container for popular self-hosted applications.

## README Template (Per Repo)

Each starter repo needs:
1. **What this is** (2 sentences)
2. **How to deploy on CLI** (step by step)
3. **What to build next** (3 project ideas)

## UI Labels

Use plain language in the deploy dialog:
- "Web App" not "Next.js"
- "AI Backend" not "FastAPI"
- "Chat Bot" not "Python Telegram Bot"
- "Simple Site" not "Static HTML"
- "API Server" not "Express.js"
- "OpenClaw" (brand name, keep as-is)

Framework name goes in the description text, not the title.
