# Container Deployment Backend — Test Scaffolding Spec

Generated by Clide | Scope: deployment API endpoint testing framework

---

## Test Framework Selection

- Language: PHP 8.2+ (matches existing CLI backend stack)
- Test runner: PHPUnit 10+
- HTTP testing: GuzzleHTTP for API endpoint simulation
- Database fixtures: SQLite in-memory for test isolation (portable, zero-config)
- CI integration: GitHub Actions — run on every PR against main

## Test Categories

### 1. Container Create Endpoint

Test case | Input | Expected
--- | --- | ---
Valid template name | POST /deploy { template: "nextjs", name: "my-app" } | 201, returns container_id, status=pending
Missing template name | POST /deploy { name: "my-app" } | 422, validation error
Invalid template name | POST /deploy { template: "nonexistent", name: "x" } | 404, template not found
Duplicate project name | POST /deploy twice with same name | 409, conflict
Name too short | POST /deploy { template: "nextjs", name: "ab" } | 422, name min length 3
Name with invalid chars | POST /deploy { template: "nextjs", name: "my app!" } | 422, sanitization error

### 2. Container Status Endpoint

Test case | Input | Expected
--- | --- | ---
Existing container | GET /deploy/{id} | 200, returns status object
Non-existent container | GET /deploy/{fake-id} | 404
Container lifecycle | Create -> poll status | Transitions: pending -> building -> running
Stopped container | GET /deploy/{stopped-id} | 200, status=stopped

### 3. Container Delete Endpoint

Test case | Input | Expected
--- | --- | ---
Running container | DELETE /deploy/{id} | 200, schedules termination
Already terminated | DELETE /deploy/{terminated-id} | 200, idempotent
Non-existent | DELETE /deploy/{fake-id} | 404

### 4. Environment Variable Handling

Test case | Input | Expected
--- | --- | ---
Valid env vars | POST /deploy { template, name, env: { KEY: "val" } } | 201, env stored
Reserved key | POST /deploy { ..., env: { PORT: "9000" } } | 422, reserved key override blocked
Too many vars | POST /deploy { ..., env: [51 vars] } | 422, limit exceeded
Secret values masked | GET /deploy/{id} | 200, env values containing "secret"/"key"/"token" are masked

### 5. Auth & Access Control

Test case | Input | Expected
--- | --- | ---
No auth header | GET /deploy | 401
Invalid token | GET /deploy with bad Bearer | 401
Valid token, different org | GET /deploy/{other-org-container} | 403
Valid token, own container | GET /deploy/{own-id} | 200
