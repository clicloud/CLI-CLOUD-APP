# Simple Site Starter — Static HTML

The absolute simplest deploy. One `index.html` file. Prove the platform works in 60 seconds.

## Deploy on CLI

1. Fork or clone this repo
2. Push to your GitHub account
3. In CLI dashboard: New Deployment → Select Repo → Deploy
4. Your site is live

## What to build next

- **Landing page** — Add sections, images, and a contact form
- **Documentation site** — Add pages and navigation
- **Web3 frontend** — Add wallet connect and onchain interactions

## Tech stack

- Nginx (static file server)
- Zero build step — just HTML, CSS, JS
