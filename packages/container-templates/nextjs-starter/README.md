# Web App Starter — Next.js

Full-stack React framework with API routes, SSR, and static generation. Deploy it on CLI in under 5 minutes.

## Deploy on CLI

1. Fork or clone this repo
2. Push to your GitHub account
3. In CLI dashboard: New Deployment → Select Repo → Deploy
4. Your app is live at `https://your-app.cli.cloud`

Or deploy via CLI terminal:
```bash
git clone https://github.com/clicloud/templates/nextjs-starter my-app
cd my-app
# Push to your repo, then deploy from CLI dashboard
```
